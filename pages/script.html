Designing Writing System

type design / script / Robofont / 2018

For the Designing Writing System we had to use 12 objects as starting point for creating our own writing system. I chose 12 various vases and simplified their shapes in order to create a fluid and usable system, which later on I digitalised with a software for type design.
The final result was exhibited in the Gallery of The Royal Academy of Arts, The Hague.
